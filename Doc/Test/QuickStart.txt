Naming stuff=Our convention is to use CamelCase for words that you RunTogether.

Naming Files=Our convention to separate words by underscore. MyDog.=[Test/Sandy_The_Dog.txt].

Bad File Names=There are a couple of things that don't work in file names.  The apostrophe, the comma  and likely others unknown are a problem. Spaces?

Naming Keys=Our convention is to separate words by periods (dots) MyDog.Color=Brown 

Bad Key Names=You can't use the opening or closing curly bracket ( { } )("brace") in a key name.
